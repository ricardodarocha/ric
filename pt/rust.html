Como funciona o curso?

A Unifagoc disponibilizou o lab0 para este curso. Os encontros serão presenciais na UNIFAGOC segundas-feiras às 19hs.

Qual o prazo de duração do curso?

4 encontros presenciais.

O curso será transmitido na modalidade EAD

Não. Este curso será unicamente presencial, porém será oferecido um repositório on-line com as atividades do curso no github

Precisa levar computador?

Não, o laboratório lab0 dispõe de 40 computadores de última geração que são suficientes para este treinamento

É necessário ter conhecimento prévio em linguagem de programação?

Não. O curso será oferecido para estudantes no primeiro período, que ainda não possuem contato com programação

Se eu tiver em um período mais avançado no curso eu posso matricular?

Sim. O curso possui uma série de links de referência que te permitirão aprender sozinho, incluindo acesso a comundiade de games, desenvolvimento WEB e nichos de pesquisa acadêmica na linguagem Rust.
